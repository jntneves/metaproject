# Meta Project - Semana Spring React
O objetivo deste projeto era construir uma aplicação completa de consulta de vendas, onde é possível buscar vendas em um dado intervalo de datas e notificar via SMS os dados dos melhores vendedores. Com isso, a ideia era desenvolver o escopo da aplicação utilizando API REST com Java/Spring, banco de dados com ORM, frontend com React/TypeScript e implantação na nuvem CI/CD.
